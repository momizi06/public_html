# public_html
一応公開用のサイトのhtml置いとく

完全に身内ネタです。

## ディレクトリについて

### /docs
公開してるデータが入ってる

### /docs/test
新しい試みをしたいけど運用を止めたく無い時にここにおいてテストする  
プルリクすると消えるので注意

### /finish_working
使い終わったファイル(過去に作ったやつとか)の置き場

### /font
フォント置き場

### /temporary
仮の書類置き場
iOS,Windows間とかでの受け渡しに使う
